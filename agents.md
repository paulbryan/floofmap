# Agents

This document provides an overview of agents used in the project, their roles, and how to interact with them.

## Available Agents

- **Plan**: Researches and outlines multi-step plans for complex tasks.
- **AIAgentExpert**: Expert in streamlining and enhancing the development of AI Agent Applications, including code generation, model comparison, tracing setup, and evaluation planning.

## Usage

Agents can be invoked to perform autonomous research, code generation, or multi-step tasks. Specify the agent name and provide a detailed prompt for best results.

## Example

To run a subagent:

- Choose the appropriate agent for your task.
- Provide a clear description and detailed prompt.
